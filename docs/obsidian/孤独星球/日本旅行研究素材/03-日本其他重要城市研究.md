# 日本深度旅行研究报告：城市/地区文化遗产全览

> 研究报告涵盖镰仓、大阪、姬路、日光、广岛/宫岛、白川乡/五箇山、富士山地区、冲绳。
> 每个地点包含历史背景、必看细节、建造年代。

---

## 一、镰仓（Kamakura）

### 1.1 镰仓大佛（高德院 Kotoku-in）

**为什么值得看**：日本第二大青铜佛像（国宝），750年来露天而坐，完整保留了13世纪原始形态，是镰仓时代佛教艺术的最高代表 [来源](https://kotoku-in.jp/en/about.html)。

**看什么细节**：
- 佛像高约11.3米，重约121吨，内部中空，可入内参观（20日元） [来源](https://jref.com/articles/kotokuin-and-the-great-buddha-of-kamakura.547/)
- 面部特征：眉间白毫（byakugō）由6公斤纯银铸造；头上656个右旋螺发；耳长1.9米 [来源](https://jref.com/articles/kotokuin-and-the-great-buddha-of-kamakura.547/)
- 耳部仍可辨认残留的金箔痕迹——最初全身镀金 [来源](https://jref.com/articles/kotokuin-and-the-great-buddha-of-kamakura.547/)
- 头部微向前倾，营造与观者的亲近感

**历史背景**：
- 建造始于1252年（镰仓时代中期），据《吾妻镜》记载，由僧侣净光募款铸造 [来源](https://kotoku-in.jp/en/about/characteristic.html)
- 最初有木造版本（1243年完成，1247年毁于暴风雨）；后改铸铜像，铸工为小野五郎右卫门 [来源](https://jref.com/articles/kotokuin-and-the-great-buddha-of-kamakura.547/)
- 原置于大佛殿中，但1334年、1369年台风及1498年明应地震引发的海啸摧毁了殿堂，此后一直露天 [来源](https://kotoku-in.jp/en/about.html)
- 江户中期由净土宗僧人祐天、养国修复；寺庙现属净土宗 [来源](https://kotoku-in.jp/en/about.html)

### 1.2 鹤冈八幡宫（Tsurugaoka Hachimangu）

**为什么值得看**：镰仓幕府的精神核心，源赖朝亲自迁建，见证了日本第一个武士政权的崛起与终结 [来源](https://nippon.com/en/japan-topics/c10910/)。

**看什么细节**：
- 本殿（1828年德川家齐重建）为流造权现造样式，是国家重要文化财 [来源](https://japan-shrine.info/en/tsurugaoka-hachimangu-shrine-kanagawa/)
- 大石段旁的最古老银杏树：1219年三代将军源实朝在此被暗杀（2010年倒伏，已萌新枝） [来源](https://nippon.com/en/japan-topics/c10910/)
- 舞殿：静御前曾在此舞蹈表达对源义经的思念 [来源](https://japan-shrine.info/en/tsurugaoka-hachimangu-shrine-kanagawa/)
- 段葛（Dankazura）：源赖朝为求子而建的樱花步道，取名"若宫大路"
- 宝物馆（200日元）：展出三基桃山时代神轿、源赖朝发髻中的小型佛像 [来源](https://japan-shrine.info/en/tsurugaoka-hachimangu-shrine-kanagawa/)

**历史背景**：
- 1063年：源赖义平定前九年之役后，从京都石清水八幡宫劝请，建于由比滨 [来源](https://samurai-archives.com/w/index.php?mobileaction=toggle_view_desktop&title=Tsurugaoka_Hachimangu)
- 1180年：源赖朝将神社迁至现址，作为镰仓城市中轴线的北端 [来源](https://visit.trip-kamakura.com/things-to-do/tsurugaoka-hachimangu/)
- 1219年：源实朝暗杀事件发生于此，标志着源氏将军世系终结
- 此后受北条氏、丰臣秀吉、德川家历代庇护 [来源](https://visit.trip-kamakura.com/things-to-do/tsurugaoka-hachimangu/)

### 1.3 建长寺（Kencho-ji）与圆觉寺（Engaku-ji）

**建长寺——日本最古老的禅宗修行道场**

**为什么值得看**：镰仓五山第一名，日本第一座纯临济宗禅寺，由南宋渡来僧兰溪道隆开创，至今保持禅宗修行传统 [来源](https://zen.rinnou.net/head_temples/04kencho.html)。

**看什么细节**：
- 三门（1754年重建）：镰仓最壮丽的禅宗山门
- 佛殿：1647年从东京增上寺移筑而来，原为德川秀忠夫人灵屋 [来源](https://jref.com/articles/kenchoji-temple.465/)
- 主尊为地藏菩萨（禅寺中极为罕见），殿内还有数百尊小型铸铁地藏
- 法堂（龙王殿）前庭园的"心字池"
- 兰溪道隆从中国带来的种子长成的700年古柏 [来源](https://zen.rinnou.net/head_temples/04kencho.html)

**历史背景**：
- 1253年由北条时赖创立，南宋高僧兰溪道隆（Rankei Doryu）任开山住持 [来源](https://en.wikipedia.org/wiki/Kenchoji)
- 兰溪所创《建长寺法语》为日本国宝 [来源](https://zen.rinnou.net/head_temples/04kencho.html)
- 曾历1293年地震、1315年和1414年大火 [来源](https://jref.com/articles/kenchoji-temple.465/)
- 19世纪在禅僧青空寒道（Aozora Kando）指导下复兴

**圆觉寺——镰仓五山第二名**

**为什么值得看**：1282年为超度元日战争双方战死者而建，舍利殿（Shariden）与大钟均为国宝 [来源](https://en.wikipedia.org/wiki/Engaku-ji)。

**看什么细节**：
- 舍利殿（国宝）：现存唯一镰仓时代唐样建筑，原为1285年建造，现存为室町时代从太平寺尼庵移筑；内供奉佛牙舍利（相传源实朝得自中国） [来源](https://en.wikipedia.org/wiki/Engaku-ji)
- 大钟（国宝）：高2.6米，镰仓最大梵钟，1301年铸造 [来源](https://en.wikipedia.org/wiki/Engaku-ji)
- 三门：1785年重建

**历史背景**：
- 1282年由中国僧无学祖元（Mugaku Sogen）受北条时宗邀请创建 [来源](https://www.engakuji.or.jp/en/about/)
- 寺名来自施工中掘出的《圆觉经》 [来源](https://en.wikipedia.org/wiki/Engaku-ji)
- 镰仓时代为对抗蒙古的精神堡垒 [来源](https://www.engakuji.or.jp/en/about/)

### 1.4 镰仓五山（Kamakura Gozan）

**简介**：镰仓五山是仿中国南宋五山十刹制度建立的禅宗寺院等级体系，由室町幕府第三代将军足利义满正式确定排名 [来源](https://en.wikipedia.org/wiki/Kenchoji)：

1. **建长寺**（第1位） — 日本最古禅宗专门道场
2. **圆觉寺**（第2位） — 元寇战殁者供养寺
3. **寿福寺**（第3位） — 源实朝之墓所在地
4. **净智寺**（第4位） — 北条氏一族菩提寺
5. **净妙寺**（第5位）

**看点**：五山系统不单是宗教中心，更是幕府的实际行政与外交机构，僧侣充任翻译、外交官、顾问 [来源](https://en.wikipedia.org/wiki/Kenchoji)。五山文学是日本中世文学的重要组成部分。

## 二、大阪（Osaka）

### 2.1 大阪城

**为什么值得看**：丰臣秀吉统一日本的象征，现存天守阁俯瞰大阪全景，石垣群是日本城郭建筑技术的顶峰 [来源](https://gov-online.go.jp/eng/publicity/book/hlj/html/202110/202110_07_en.html)。

**看什么细节**：
- 现存天守阁：1931年市民捐款重建，钢筋混凝土结构，外观5层实为8层（含地下），高58米 [来源](https://www.mlit.go.jp/tagengo-db/en/R1-01758.html)
- 白绿相间的外观与金饰（金鯱 shachihoko） [来源](https://osakacastle.org/)
- **蛸石**（Tako-ishi）：日本最大的单体城石，高5.5米、宽11.7米、重约108吨 [来源](https://gov-online.go.jp/eng/publicity/book/hlj/html/202110/202110_07_en.html)
- 石垣来自九州、濑户内海等地，总长约12公里，使用约100万吨花岗岩 [来源](https://gov-online.go.jp/eng/publicity/book/hlj/html/202110/202110_07_en.html)
- 大手门附近三块巨型石板来自同一巨石切割 [来源](https://gov-online.go.jp/eng/publicity/book/hlj/html/202110/202110_07_en.html)
- 城内博物馆展示武士铠甲、武器和丰臣时代模型 [来源](https://osakacastle.org/)

**历史背景**：
- 1583年：丰臣秀吉在石山本愿寺旧址上开始筑城，1597年落成 [来源](https://osakacastle.org/history/)
- 1614-1615年：大阪之阵中几乎全毁；1620年德川幕府重建（扩大石垣规模为原先两倍） [来源](https://www.city.osaka.lg.jp/contents/wdu020/kensetsu/english/rekishi/uekita/p49to50e.htm)
- 1665年：天守阁遭雷击烧毁，此后250余年间无天守
- 1931年：第三次天守阁重建，基于黑田屏风《大阪夏之阵图》等史料 [来源](https://www.mlit.go.jp/tagengo-db/en/R1-01758.html)
- 园内现存13栋江户时代建筑为重要文化财（大手门、樱门、各种橹等） [来源](https://www.city.osaka.lg.jp/contents/wdu020/kensetsu/english/rekishi/uekita/p49to50e.htm)

### 2.2 四天王寺（Shitennoji）

**为什么值得看**：日本最古老的国家官刹，由圣德太子于593年创建，四天王寺式伽蓝配置是飞鸟时代佛教建筑的珍贵遗存 [来源](https://www.mlit.go.jp/tagengo-db/en/R4-00628.html)。

**看什么细节**：
- 伽蓝配置：中门→五重塔→金堂→讲堂在同一南北直线上排列——即"四天王寺式伽蓝" [来源](https://www.city.osaka.lg.jp/contents/wdu020/kensetsu/english/rekishi/uekita/p76e.htm)
- 五重塔：可登临，现建筑为1965年重建
- 金堂：供奉圣德太子化身的观音像及本尊四天王像；壁面描绘释迦一生 [来源](https://www.mlit.go.jp/tagengo-db/en/R4-00628.html)
- **石鸟居**（1294年敕建）：日本最古老的石鸟居，重要文化财；铜板铭文称此为"极乐之东门"——13世纪时站在此可望见大阪湾的落日，引发净土信仰的西方极乐联想 [来源](https://www.japan-experience.com/all-about-japan/osaka/temples-shrines/shitennoji-temple-tennoji-osaka)
- 六时堂、本坊庭园（极乐净土式庭园）、猫门等 [来源](https://www.mlit.go.jp/tagengo-db/en/R4-00628.html)
- 每月21-22日有大师会/太子会，设众多露天摊位 [来源](https://en.osaka-info.jp/spot/shitennoji/)
- 宝物馆：藏有国宝"扇面法华经册子"等 [来源](https://www.japan-experience.com/all-about-japan/osaka/temples-shrines/shitennoji-temple-tennoji-osaka)

**历史背景**：
- 593年（推古天皇元年）：圣德太子创建，为纪念四天王庇佑圣德太子战胜物部氏 [来源](https://www.japan-guide.com/e/e4011.html)
- 屡遭火灾和战乱，最近一次主建筑重建为1965年（昭和时期） [来源](https://www.mlit.go.jp/tagengo-db/en/R4-00628.html)
- 圣德太子被神化为救世观音，金堂供奉其等身大像
- 为和宗总本山

### 2.3 住吉大社（Sumiyoshi Taisha）

**为什么值得看**：日本最古老神社之一，以"住吉造"纯日本本土神社建筑样式著称——最古老的神社建筑样式之一，四栋国宝本殿面西向海而立 [来源](https://en.wikipedia.org/wiki/Sumiyoshi-zukuri)。

**看什么细节**：
- **住吉造**（Sumiyoshi-zukuri）特征：妻入式（入口设在山墙侧）、直坡切妻屋顶（屋檐不上翘）、桧皮葺（桧树皮多层葺顶）、丹土涂（朱红柱）与胡粉涂（白壁）、屋顶设千木（forked finial）与五根坚鱼木 [来源](https://en.wikipedia.org/wiki/Sumiyoshi-zukuri)
- 现存本殿建于1810年，但忠实保持了佛教传来前的古制 [来源](https://gov-online.go.jp/eng/publicity/book/hlj/html/202101/202101_13_en.html)
- 四栋本殿排列呈船队形，面西朝向大阪湾（与绝大多数神社面南/面东迥异） [来源](https://gov-online.go.jp/eng/publicity/book/hlj/html/202101/202101_13_en.html)
- 角鸟居（Kakutorii）：住吉式独特鸟居——笠木两端不伸出柱外，全部构件为方形截面 [来源](https://en.wikipedia.org/wiki/Sumiyoshi_taisha)
- 太鼓桥（反桥）、重軽石（Omokaruseki）等特色景点 [来源](https://nihonmono.jp/en/article/30979/)
- 600余基石灯笼多为江户时代航运业者捐赠，曾用作航海灯台 [来源](https://gov-online.go.jp/eng/publicity/book/hlj/html/202101/202101_13_en.html)

**历史背景**：
- 传统记载建于公元211年（应神天皇时期），由田裳见宿祢（Tamomi no Sukune）奉神功皇后命创建 [来源](https://en.wikipedia.org/wiki/Sumiyoshi_taisha)
- 为全国约2300家住吉神社的总本社 [来源](https://nihonmono.jp/en/article/30979/)
- 7-9世纪，遣隋使/遣唐使出发前在此祈祷航海安全
- 四栋本殿分别祭祀住吉三神（海底/中/表筒男命）和神功皇后

### 2.4 国立国际美术馆（NMAO）

**为什么值得看**：世界罕见的全地下美术馆，由西萨·佩里设计，专注1945年以来的近现代艺术，馆藏约8200件 [来源](https://www.nmao.go.jp/en/about/architecture-and-design/?lang=en)。

**看什么细节**：
- 建筑：竹的生命力为灵感，地上仅见入口钢构装置，三阶地下空间由玻璃天井自然采光的挑空贯穿 [来源](https://www.nmao.go.jp/en/about/architecture-and-design/?lang=en)
- B1层常设琼·米罗陶瓷壁画《天真的笑声》（1969）、亚历山大·考尔德雕塑《伦敦》（1962）等 [来源](https://www.nmao.go.jp/en/about/architecture-and-design/?lang=en)
- 展室地板为纵切花旗松木块（吸音减噪），挑空壁材为米色方钠石 [来源](https://www.nmao.go.jp/en/about/architecture-and-design/?lang=en)

**历史背景**：
- 前身为1970年大阪世博会美术馆；现址于2004年在中之岛开业 [来源](https://www.nmao.go.jp/en/about/history/?lang=en)
- 日本唯一的国立专业近现代美术馆
- 建筑师西萨·佩里（阿根廷裔美国建筑师，亦为吉隆坡双子塔设计师）

## 三、姬路（Himeji）

### 3.1 姬路城（Himeji-jo / 白鹭城）

**为什么值得看**：日本保存最完好的城堡，UNESCO世界遗产（1993年登录），国宝。白色灰泥墙面营造出"白鹭展翅"的优雅形态，17世纪木造城郭建筑的巅峰 [来源](https://whc.unesco.org/en/list/661/)。

**看什么细节**：
- 大天守（Dai-Tenshu）：外观5层实为6层+地下室，高约33米 [来源](https://web-japan.org/atlas/architecture/arc16.html)
- 由大天守+三小天守+连接渡橹构成的天守群 [来源](https://whc.unesco.org/document/153975)
- 白色石灰膏壁统合全部建筑体，形成统一美感 [来源](https://whc.unesco.org/en/list/661/)
- 迷宫式通路：从菱门进入后路径曲折复杂——这是有意设计的防御机制 [来源](https://faq.japan-travel.jnto.go.jp/en/spot/1030/)
- 保留完整的内堀（可乘船游览，周末节假日限定，1000日元） [来源](https://faq.japan-travel.jnto.go.jp/en/spot/1030/)
- 天守最顶层设小神社，可俯瞰城市全景 [来源](https://faq.japan-travel.jnto.go.jp/en/spot/1030/)
- 1000余株樱花，春季为赏樱名所 [来源](https://faq.japan-travel.jnto.go.jp/en/spot/1030/)
- 紧邻的好古园（Kokoen）：1992年开园，九个小庭园分别呈现不同江户样式 [来源](https://faq.japan-travel.jnto.go.jp/en/spot/1030/)

**历史背景**：
- 1346年：最初筑城 [来源](https://web-japan.org/atlas/architecture/arc16.html)
- 1580年：丰臣秀吉重建
- 1601-1609年：池田辉政大规模扩建，形成今日面貌，共含80余栋建筑 [来源](https://whc.unesco.org/document/153975)
- 1617年：本多忠政为妻子（德川家康之女）改建西之丸 [来源](https://whc.unesco.org/document/153975)
- 1995年阪神大地震中天守几乎毫发无损——连顶上神社供桌上的酒瓶都未倒 [来源](https://web-japan.org/atlas/architecture/arc16.html)
- 2015年完成长达5年的大规模平成大修理
- 日本现存12天守（未被战火/灾害毁灭的木造天守）中保存最完整者

## 四、日光（Nikko）

> 日光山内（二社一寺）：1999年UNESCO登录"日光社寺"，包含日光东照宫、轮王寺、二荒山神社共103栋建筑，其中9件国宝、94件重要文化财 [来源](https://whc.unesco.org/en/list/661/)。

### 4.1 日光东照宫（Nikko Toshogu）

**为什么值得看**：德川家康的灵庙，日本最华丽的神社建筑群。240,000枚金箔覆盖的阳明门被称为"日本最美的门"，500余尊雕刻包含"不看、不言、不闻"三猿 [来源](https://www.gov-online.go.jp/eng/publicity/book/hlj/html/202210/202210_03_en.html)。

**看什么细节**：
- **阳明门**（国宝）：宽约7米、深4米、高11米，装饰508尊雕刻——龙、凤凰、中国圣贤故事、儿童嬉戏等，因终日看不厌而别名"日暮门"（Higurashimon） [来源](https://www.gov-online.go.jp/eng/publicity/book/hlj/html/202210/202210_03_en.html)
  - 著名雕刻"司马光砸缸"——传达"人命重于一切"之理念 [来源](https://www.gov-online.go.jp/eng/publicity/book/hlj/html/202210/202210_03_en.html)
- **三猿雕刻**（神厩舍）：不看、不言、不闻
- **眠猫**（Nemuri-neko）：传为左甚五郎作品，在通往奥社走廊门上
- **唐门**（Karamon）：白胡粉涂的精致门
- **奥社**（Okumiya）：家康墓所——穿过207级石阶的杉木林道，气氛肃穆简约（有意与下方华丽形成对比） [来源](https://www.nippon.com/en/guide-to-japan/gu900202/)
- 石鸟居（1618年建）：高9.2米、宽13.2米，笠木中空以减重抗震 [来源](https://www.city.nikko.lg.jp/soshiki/10/1041/1_1/2/6/1393.html)

**历史背景**：
- 1617年：二代将军秀忠造营，初代东照社 [来源](https://www.city.nikko.lg.jp/soshiki/10/1041/1_1/2/6/1393.html)
- 1636年：三代将军家光进行大改造（宽永大造替），网罗全国最高级工匠，将神社扩建成今日的华丽面貌 [来源](https://www.nippon.com/en/guide-to-japan/gu900202/)
- 一反神道神社素简传统，以金箔浓彩彰显幕府权威 [来源](https://www.ohajapan.com/tochigi/nikko-toshogu-shrine/)
- 家康被神化为"东照大权现"
- 近年完成6年大修（2019年完工），恢复了创建时的辉煌 [来源](https://www.nippon.com/en/guide-to-japan/gu900202/)

### 4.2 日光山轮王寺（Rinnoji）

**为什么值得看**：日光最古的佛教寺院（创建于766年），主堂三佛堂内部供奉三尊高达8米的金色木造大佛，德川家光的大猷院灵庙为国宝 [来源](https://www.rinnoji.or.jp/en/history.html)。

**看什么细节**：
- **三佛堂**（Sanbutsudo）：日光最大木造建筑，内供千手观音（男体山化身）、阿弥陀如来（女峰山化身）、马头观音（太郎山化身）——佛教诸佛对应三座神山 [来源](https://www.rinnoji.or.jp/en/spot/)
- **大猷院**（Taiyuin，国宝）：三代将军家光的灵庙——刻意以金黑配色较东照宫减抑，以表达对祖父的敬意 [来源](https://www.rinnoji.or.jp/en/spot/02.html)
  - 三座门各有两尊以上仁王守护（常规仅一座），沿途石灯笼和铜灯笼由各地大名捐献以示臣服 [来源](https://www.rinnoji.or.jp/en/spot/02.html)
- 宝物殿（宝藏馆）：收藏国宝级经卷和佛像
- 逍遥园（Shoyoen）：池泉回游式日本庭园，秋季红叶名所 [来源](https://www.japan-guide.com/e/e3803.html)

**历史背景**：
- 766年（天平神护二年）：开山僧胜道上人（Shodo Shonin, 735-817）在日光开创 [来源](https://www.rinnoji.or.jp/en/history.html)
- 1613年：德川家康任命天海僧正为住持，轮王寺进入鼎盛期——天海策划了东照宫建造和家康神格化 [来源](https://www.rinnoji.or.jp/en/history.html)
- 1871年神佛分离令后，原"日光山"分离为今日的二社一寺体制 [来源](https://www.rinnoji.or.jp/en/history.html)

### 4.3 二荒山神社（Futarasan Shrine）

**为什么值得看**：日光山岳信仰的起源，创立于782年，祭祀日光三山（男体山、女峰山、太郎山）——是"日光"地名的由来 [来源](https://www.mlit.go.jp/tagengo-db/en/R2-00015.html)。

**看什么细节**：
- 本殿（重要文化财）：1619年由二代将军德川秀忠寄进，祭祀大国主命、田心姬命、味耜高彦根命 [来源](https://www.mlit.go.jp/tagengo-db/en/H30-00257.html)
- **神桥**（Shinkyo）：朱漆木造拱桥，位列日本三奇桥之一，二荒山神社入口标志
- 六面青铜灯笼（1292年铭）：有"妖刀灯笼"之称——夜间的怪异光影令路过的武士拔刀砍之，留下70余道刀痕 [来源](https://www.mlit.go.jp/tagengo-db/en/R2-00015.html)
- 三社体制：奥宫（男体山顶）、中宫祠（中禅寺湖畔）、本社（神桥附近）

## 五、广岛 / 宫岛（Hiroshima / Miyajima）

### 5.1 严岛神社（Itsukushima Shrine）

**为什么值得看**：日本三景之一，1996年UNESCO世界遗产。寝殿造（shinden-zukuri）应用于神社建筑的世界孤例，满潮时朱红色社殿群"浮于"濑户内海上——人工建筑、海洋、群山的完美和谐 [来源](https://whc.unesco.org/en/list/776/)。

**看什么细节**：
- **大鸟居**（Otorii）：高约16米、宽24米，日本最大级木造鸟居之一；距本社约160米，建于海中。满潮时如浮于水上，退潮时可步行至其下。1875年重建（第九代），2022年12月完成约3.5年大修——桧皮葺屋顶全面替换 [来源](https://gov-online.go.jp/eng/publicity/book/hlj/html/202304/202304_02_en.html)
- 社殿群：总长约260米的东西回廊连接，廊下木板间留缝——涨潮时海水可从缝隙溢出，减少浮力冲击 [来源](https://gov-online.go.jp/eng/publicity/book/hlj/html/202304/202304_02_en.html)
- 能舞台：日本三名舞台之一，在海上演出雅乐 [来源](https://www.japan.travel/en/world-heritage/itsukushima-shinto-shrine/)
- 五重塔、多宝塔
- **平家纳经**（Heike Nokyo）：平清盛一族奉献的装饰经卷，国宝 [来源](https://japan-stroll.com/miyajima/english/guide/history-culture/itsukushima-architecture-history/)

**历史背景**：
- 593年：据传由佐伯鞍职奉神谕创建 [来源](https://www.itsukushimajinja.jp/en-sp/history.html)
- 1168年：平清盛大规模扩建，采用寝殿造（平安贵族宅邸样式），奠定了今日格局 [来源](https://japan-stroll.com/miyajima/english/guide/history-culture/itsukushima-architecture-history/)
- 此后经历两次镰仓时代大火，现存建筑主要为1240-1243年重建 [来源](https://japan-stroll.com/miyajima/english/guide/history-culture/itsukushima-architecture-history/)
- 全岛因被视为神体（神域），历史上禁止一般人居住，诞生死者的场合须迁往对岸
- 弥山（Misen）山麓原始林为自然遗产，自古为山岳信仰圣地

### 5.2 广岛和平纪念资料馆 & 原爆Dome

**为什么值得看**：1996年UNESCO世界遗产。人类历史上首次核武器使用的唯一幸存建筑物，保存为"核武器废弃与永久和平"的象征 [来源](https://whc.unesco.org/en/list/775/)。

**看什么细节**：
- **原爆Dome**：原为"广岛县产业奖励馆"（1915年落成，捷克建筑师Jan Letzel设计），距爆心约160米——因爆风几乎垂直直击而中心墙体奇迹幸存 [来源](https://gov-online.go.jp/eng/publicity/book/hlj/html/202304/202304_03_en.html)
  - 自1967年首次大规模保存工事以来，每三年进行一次详细调查，使用钢材与合成树脂最小限度加固以"保持爆炸后状态" [来源](https://whc.unesco.org/en/list/775/)
- **和平纪念资料馆**：1955年开馆，丹下健三设计，2006年成为日本首个战后建筑重要文化财 [来源](https://www.fun-japan.jp/en/articles/9301)
  - 2019年全面改装，主馆展示受难者遗物与照片，东馆展示核时代历史 [来源](https://en.wikipedia.org/wiki/Hiroshima_Peace_Memorial_Museum)
  - 馆藏超20,000件原爆资料 [来源](https://www.fun-japan.jp/en/articles/9301)
- **和平纪念公园**（122,100 m²）：含原爆死没者慰灵碑等 [来源](https://www.fun-japan.jp/en/articles/9301)

**历史背景**：
- 1945年8月6日8:15：原子弹在广岛市上空约600米处爆炸，截至当年度末约14万人死亡 [来源](https://www.fun-japan.jp/en/articles/9301)
- 战后关于是否保存遗址曾引发激烈争论（部分市民希望拆除以忘却创伤） [来源](https://www.city.hiroshima.lg.jp/english/peace/1029869/1010013.html)
- 1996年：经165万余人署名运动推动，原爆Dome登录为世界遗产 [来源](https://www.city.hiroshima.lg.jp/english/peace/1029869/1010013.html)

## 六、白川乡 / 五箇山（Shirakawa-go / Gokayama）

### 6.1 合掌造聚落

**为什么值得看**：1995年UNESCO世界遗产。日本唯一的Gassho-zukuri（合掌造）民居群——名称源于屋顶形如双手合十祈祷。此建筑样式仅见于白川乡和五箇山地区，是人类适应豪雪环境的极致智慧 [来源](https://whc-shirakawa-goandgokayama.jp/en/value/)。

**看什么细节**：
- **屋顶结构**：茅草葺切妻屋顶，倾斜度约60度——使积雪（有时深达4米）易于滑落 [来源](https://www.japan.travel/en/world-heritage/the-historic-villages-of-shirakawa-go-and-gokayama/)
- **阁楼利用**：屋顶内部被分为2-4层用于养蚕（Sericulture）——几乎所有日本传统农家的屋顶空间都废弃不用，唯独合掌造将阁楼作为工作空间 [来源](https://whc-shirakawa-goandgokayama.jp/en/value/)
- **结构特点**：合掌构造（rafter-frame）原本会导致结构弱点，但通过"将整体屋面板连为一体"的独特技法克服——此技术在别处没有 [来源](https://whc-shirakawa-goandgokayama.jp/en/value/)
- **无需钉子**：茅草和木材通过绳索捆绑
- **地域差异** [来源](https://whc-shirakawa-goandgokayama.jp/en/value/)：
  - 白川乡：妻入式（gable-side entrance），山墙斜倾
  - 五箇山：平入式（front entrance），山墙附庇（eaves），内部保留土间
- **三个指定集落** [来源](https://whc.unesco.org/en/list/734/)：
  - **荻町**（Ogimachi，白川乡）：59栋合掌造，最大聚落。可从荻町城迹展望台俯瞰全村。可入和田家、长濑家参观 [来源](https://www.japan.travel/en/world-heritage/the-historic-villages-of-shirakawa-go-and-gokayama/)
  - **相仓**（Ainokura，五箇山）：20栋，群山环绕。村上家可由家长围炉讲史
  - **菅沼**（Suganuma，五箇山）：9栋，五箇山最大合掌造岩濑家（5层，榉木构）

**历史背景**：
- 该地区长期被称为"日本最后的秘境"——1950年代以前因豪雪和险峻地形几乎与外界隔绝 [来源](https://en.wikipedia.org/wiki/Historic_Villages_of_Shirakawa-g%C5%8D_and_Gokayama)
- 合掌造在19世纪末鼎盛期约有1850户，而当时日本全国有550万户农家——占比仅0.03% [来源](https://whc-shirakawa-goandgokayama.jp/en/value/)
- 集落间由庄川水运连接，形成独特的真宗信仰文化圈 [来源](https://whc-shirakawa-goandgokayama.jp/en/value/)
- 现存最古建筑可追溯至17世纪末（江户时代初期）

## 七、富士山地区（Mount Fuji）

### 7.1 富士山本身

**为什么值得看**：2013年UNESCO世界文化遗产（登录名"富士山——信仰的对象与艺术的源泉"）。海拔3,776米的独立成层火山，日本最高峰，其近乎完美的圆锥形影响了日本的宗教、艺术（尤其是葛饰北斋《富岳三十六景》给西方艺术带来冲击） [来源](https://whc.unesco.org/en/list/1418/)。

**看什么细节**：
- 世界遗产涵盖25个构成资产：登山道、山顶火山口神社群、山麓的浅间神社群、御师宿坊、熔岩树型、湖沼、瀑布等 [来源](https://whc.unesco.org/en/list/1418/)
- 江户时代起登山者将山岳分为三个区域：草木域→森林域→"烧山"域（秃顶部分）
- **お鉢巡り**（Ohachimeguri）：绕火山口一周的巡礼仪式
- 7-8月开山，山顶设有富士山本宫浅间大社的奥宫 [来源](https://www.nippon.com/en/guide-to-japan/gu009003/japan%E2%80%99s-highest-shrine-fujisan-hongu-sengen-taisha.html)

**信仰背景**：
- 自12世纪起成为修验道（融合神道和佛教的山岳修行）中心 [来源](https://whc.unesco.org/en/list/1418/)
- **富士讲**（Fuji-ko）：江户时代兴起的民间富士信仰组织，团体登拜盛行
- **富士山本宫浅间大社**：全国约1300社的总本宫。德川家康在关原之战后捐赠建造30余栋建筑，并将富士山八合目以上土地划归该社所有（至今仍有山顶所有权） [来源](https://www.nippon.com/en/guide-to-japan/gu009003/japan%E2%80%99s-highest-shrine-fujisan-hongu-sengen-taisha.html)
- 木花咲耶姬（Konohana no Sakuya Hime）为神社祭神

### 7.2 忍野八海（Oshino Hakkai）

**为什么值得看**：富士山世界遗产的构成资产之一，八个清澈见底的涌泉池——水来自富士山降雪融水在地下过滤长达20-80年后的自然涌出 [来源](https://gov-online.go.jp/eng/publicity/book/hlj/html/202301/202301_04_en.html)。

**看什么细节**：
- 八个指定池 [来源](https://en.kawaguchiko.net/cultural-assets-en/oshinohakkai/)：出口池、釜池、底无池、铫子池、涌池、浊池、镜池、菖蒲池
- 镜池（Kagami-ike）：无风时能倒映富士山
- 涌池（Waku-ike）：八个池中涌水量最大
- 富士讲修行者登山前在此行水垢离（冷水沐浴净身） [来源](https://gov-online.go.jp/eng/publicity/book/hlj/html/202301/202301_04_en.html)
- 铫子池、底无池位于榛之木林资料馆内（有收费小型露天博物馆，展示古民家、农具、武士武器） [来源](https://www.japan-guide.com/e/e6915.html)
- 注意：中央的"中池"是人工池（在土产店旁），非八海之一 [来源](https://en.kawaguchiko.net/cultural-assets-en/oshinohakkai/)

**地理背景**：
- 约6000年前，此区域原为宇津湖；富士山延历喷发时熔岩流填平了湖盆 [来源](https://en.japanopia-fuji.com/report-oshinohakkai/)
- 1934年指定为国家天然纪念物 [来源](https://gov-online.go.jp/eng/publicity/book/hlj/html/202301/202301_04_en.html)

### 7.3 河口湖（Kawaguchiko）

**为什么值得看**：富士五湖中最易抵达、设施最完备的湖，可同时欣赏富士山倒影、享受温泉、参观世界级美术馆 [来源](https://japanandmore.com/destinations/lake-kawaguchi/)。

**看什么细节**：
- **新仓山浅间神社 忠灵塔**（Chureito Pagoda）：五重塔+富士山的经典构图，摄影师圣地。1963年建，从神社主殿攀登近400级台阶可达。樱花季（4月中旬）和红叶季（11月上旬）最佳 [来源](https://www.japan-guide.com/e/e6917.html)
- **久保田一竹美术馆**（Itchiku Kubota Art Museum）：纺织艺术家久保田一竹（1917-2003）的私人美术馆。他毕生致力于复活失传的室町时代染织技法"辻花染"（Tsujigahana）。展品包括未完成的巨作"光之交响曲"——由80件着物组成一幅富士山全景。建筑由冲绳珊瑚石灰岩和巨大木构架建造，庭园由艺术家自设计 [来源](https://www.japan-guide.com/e/e6912.html)
- **天上山公园缆车**（Mt. Kachi Kachi Ropeway）：登顶可一览富士山与河口湖全貌
- **河口湖温泉**：可边泡温泉边眺富士

## 八、冲绳（Okinawa）

### 8.1 首里城（Shuri Castle）

**为什么值得看**：琉球王国（1429-1879）的政治、外交、文化中心，2000年UNESCO世界遗产"琉球王国Gusuku及关联遗产群"的核心。建筑融合中国与日本样式，使用鲜红涂装——与本土日本城郭截然不同的视觉风格 [来源](https://oki-park.jp/shurijo/en/about/)。

**看什么细节**：
- **正殿**（Seiden）：琉球最大的木造建筑，朱红色；前方面西——这是首里城的独特特征（中国皇帝来自西方，因此以朝西以示尊敬） [来源](https://oki-park.jp/shurijo/en/about/)
- **御庭**（Una Plaza）：正殿前广场，国家仪式场所
- **守礼门**（Shureimon Gate）：首里城的标志性门，"守礼之邦"匾额见证了琉球王国以礼立国的外交传统
- 石垣：曲线形城墙环绕——这是琉球Gusuku（城）的特征，不同于本土日本城堡的直线石垣
- 目前状态：2019年10月31日火灾烧毁正殿等7栋建筑，重建工作进行中——正殿外装已于2025年完成，计划2026年秋全面内向完工 [来源](https://www.japantimes.co.jp/news/2025/10/31/japan/shuri-castle-reconstruction/)
  - 重建严格使用1712年图纸和照片，并增设消防设备和自动喷淋系统 [来源](https://www.scmp.com/week-asia/lifestyle-culture/article/3107780/one-year-after-okinawas-historic-shuri-castle-was)

**历史背景**：
- 建于14世纪末；1429年尚巴志统一琉球后成为王宫 [来源](https://www.discover-okinawa.com/about-okinawa/destinations/okinawa-main-island/shuri-castle/)
- 曾5次被毁：火灾/战争/最后的1945年冲绳战役 [来源](https://www.discover-okinawa.com/about-okinawa/destinations/okinawa-main-island/shuri-castle/)
- 1992年：基于历史记录完成战后首次重建，作为国定公园开放
- 2019年火灾烧毁4200m²建筑，包括1992年复原的所有珍贵文物 [来源](https://www.scmp.com/week-asia/lifestyle-culture/article/3107780/one-year-after-okinawas-historic-shuri-castle-was)

### 8.2 琉球王国遗址群（Gusuku Sites）

**为什么值得看**：2000年UNESCO世界遗产，由9个构成资产组成，展现了12-17世纪琉球王国500年历史的独特文化——融合中国、日本、朝鲜、东南亚的海上贸易文明 [来源](https://whc.unesco.org/en/list/972/)。

**9个构成资产一览** [来源](https://whc.unesco.org/en/list/972/)：

| 资产 | 定位 |
|------|------|
| **首里城迹** | 王国政治中心，国王居所与行政机构 |
| **玉陵**（Tamaudun） | 第二尚氏王族陵墓，巨大的石造建筑，内部铺珊瑚砂，三室分藏历代国王遗骨 [来源](https://www.japan.travel/en/world-heritage/gusuku-sites-and-related-properties-of-the-kingdom-of-ryukyu/) |
| **园比屋武御岳石門**（Sonohyan-utaki Ishimon） | 琉球国王外出时祈祷平安的石造门 |
| **中城城迹**（Nakagusuku） | 琉球保存最完好的城迹，多郭构造清晰可辨，丘陵顶视野极佳 [来源](https://www.japan-guide.com/e/e7125.html) |
| **胜连城迹**（Katsuren） | 15世纪著名按司阿麻和利的居城，海崖上的关隘状城 |
| **座喜味城迹**（Zakimi） | 坐拥残波岬附近丘陵，防御北方势力的前哨 |
| **今归仁城迹**（Nakijin） | 三山时代北山王国之王城，荒寂原始森林中的石垣群 |
| **识名园**（Shikinaen） | 琉球王家别邸与迎宾庭园，池泉回游式 |
| **斋场御岳**（Seifa-utaki） | 琉球王国最高圣地，王国时代由神女（闻得大君）主持祭祀 |

**共通看点**：
- Gusuku（城）的特征：曲线形石垣包围高台，与本土日本城堡的直线型完全不同
- 石材使用了琉球石灰岩（珊瑚礁隆起形成）
- 每座城均能眺望海洋——琉球王国的海洋文明特性
- 与首里城一同入选，构成完整的王国政治、军事、宗教体系
